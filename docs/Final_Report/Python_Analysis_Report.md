# Python Analysis Report

## Purpose
This report summarizes insights generated using Python for the PulseIQ customer experience dataset.

## Analyses Completed
- Revenue analysis
- Product sales analysis
- Returns analysis
- Customer sentiment analysis
- Support ticket analysis

## Key Insights
- Total revenue was $828.
- Basic T-Shirt was the top-selling product by quantity.
- Product Quality Issue generated the highest refund amount.
- Negative sentiment represented 40% of customer feedback.
- Support tickets were analyzed by issue type.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code
- Jupyter Notebook

## Business Recommendation
PulseStyle should focus on improving product quality, reducing refund-related issues, and improving customer satisfaction because negative sentiment represented the largest feedback category.
